Chess Game Analysis —
Machine Learning with Kaggle

This project analyses chess games sourced from Lichess and Chess.com using machine learning techniques on the Kaggle platform. The goal is to identify patterns in gameplay — covering opening strategies, common move sequences, and player performance metrics.

Multiple ML algorithms were applied to the dataset to extract insights, alongside rich visualisations to make the findings accessible and understandable. A great opportunity to combine a love of chess with data science.
